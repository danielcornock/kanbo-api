The backend for a simple kanban board application inspired by Trello. Features include:

- Linking github commits.
- Tracked story numbers.
- Custom tags & labels.
- Drag and drop stories.

> This project is a WIP.

## Technologies used

- NestJS
- TypeScript
- NodeJS
- Jest
- Express
